Webapplication for editing TV-Anytime metadata
This application provides a graphical browser based userinterface to persiste metadata defined by the latest TV-Anytime standard.

Those TV-Anytime specifications are:

SP003 Metadata (normative), version 1.3
SP003 Metadata - Corrigencda 2 to S-3 v1.3, 21/03/2003
SP004 Content Referencing (Normative), v1.2
SP004 Content Referencing - Corrigenda 1 to SP004 v1.2

The application is based on the BBC R&D TV-Anytime Java API.
The API is described by the White Paper WHP 060, T. Ferene, T. Sargeant and C. Akanbi

Manipulate Content Reference Information - CRID
Manipulate Segment Information for a multimedia content
Manipulate Program Information for a multimedia content
Add Tv-Anytime metadata as an xml-file to the database
